Pasos necesarios para ejecutar la API y acceder a su documentacion en el navegador

- clonar repositorio y ubicarse en carpeta API
- crear entorno virtual
- activar entorno virtual
- abrir terminal en carpeta API
- realizar pip install -r requirements.txt
- lanzar aplicacion con el comando: uvicorn main:app --reload
- para visualizar la documentacion abrir navegador web y navegar a localhost:8000/docs
